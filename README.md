# Famindex

Famindex project

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

```sh

npm install @headlessui/vue@latest

```



## heroicons:

```sh

npm install @heroicons/vue

```
